- #### IV. **_(Adv.)_**
    ```py
    def func(name, *subject_marks):
        print(f"Hello, {name}!")

        for subject in subject_marks:
            print(f"One of your subject marks is {subject}.")

    func("John", 75, 92, 58, 86, 63)
    ```

- #### V. **_(Adv.)_**
    ```py
    def func(business_name, **customers):
        print(f"Welcome to our {business_name} business!\n")

        print("Pending customer orders:")

            print(f"{customer} : {order}")

    func("Clothes", Magan=32, Chhagan=54, Lagan=67)
    ```
