#### Q1.

What will be the output of the following code?

<Admonition variant="caution" title="caution">
    You have to predict the output yourself without running/executing the code.
</Admonition>

<div class="w-full my-8"></div>

```jsx
let greet = "hello";

function outerGreet() {
    let greet = "hi";

    console.log(greet);

    function innerGreet() {
        console.log(greet);
    }
}

console.log(greet);

outerGreet();
```
