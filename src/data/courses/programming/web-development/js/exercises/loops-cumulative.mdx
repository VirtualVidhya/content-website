#### Q2. 

Print the factorial of a given number n.

<Admonition variant="note" title="note">
    Factorial of a number n is the product of all positive integers less than or equal to a given positive integer and denoted by that integer.
</Admonition>

<div class="w-full my-8"></div>

```ansi
[2;3m// ! is a sign of factorial [0m

5! = 120 

[2;3m// (factorial of 5) = 5*4*3*2*1 = 120

// Note that 0! is always 1, not 0[0m
```
